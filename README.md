## 📊 现金流压力测试工具 · 简易使用说明

### 简介
这是一款面向个人的按日现金流模拟工具，帮你直观看到未来余额走势、资金枯竭风险，以及存钱目标的达成时间。通过设置每日生活费、周期性收支（工资、房租等）和临时收支，工具会自动生成未来两年的每日现金流预测。

### 快速开始
1. 在顶部 **数据与设置卡** 中输入你当前的 **可用余额** 和 **计算起始日**（默认今天）。
2. 展开 **📋 收支计划**，设置你的日常开销和固定收支。
3. 观察 **📌 财务脉搏横条** 和 **🛡️ 生存底线** 卡片，立刻看到风险等级和枯竭预警。
4. 在 **🏦 财富目标** 中输入存钱金额，查看预计达成日期。

所有修改都会 **自动实时更新** 图表和风险数据，无需手动刷新。

### 界面功能说明

#### ① 数据与设置卡
- **可用余额**：你当前的现金总额。
- **计算起始日**：从哪一天开始模拟（通常为今天）。
- **备份全部计划 / 恢复计划**：导出当前所有设置为一个 JSON 文件，或导入之前备份的配置。

#### ② 财务脉搏横条
页面顶部固定显示的彩色横条，根据风险等级变色：
- 🟢 绿色：安全
- 🟡 黄色：一般危机（30天内枯竭）
- 🟠 橙色：危险（14天内枯竭）
- 🔴 红色：极度危险（7天内枯竭），**伴有脉冲动画**。

横条上还会显示当前余额、风险状态，以及你设置的存钱目标进度。

#### ③ 收支计划（可折叠）
- **🍔 每日基础开销**：设置工作日和周末的日均必要支出（如餐饮、交通）。
- **📅 固定周期计划**：添加周期性收支，支持 **每周/每月/每季度/每年** 重复。可设置：
  - 开始/结束日期
  - 提前结清日期（如提前还清贷款）
  - 启用/停用开关
- **✏️ 临时事件**：记录一次性收支（如聚餐、修车、奖金）。

#### ④ 生存底线（左卡片）
- 展示未来14天的最低余额、预计耗尽日、距离耗尽天数。
- **自定义预警线**：输入天数，点击“设为预警”，风险等级将基于该天数重新判定。
- **检测安全垫**：输入你想保障的天数，工具会计算出 **至少需要多少余额** 才不会枯竭。

#### ⑤ 财富目标（右卡片）
- 输入你想存够的金额，自动显示 **当前进度条** 和 **预计达成日期**。
- 如果未来两年内无法达成，会提示 **最高可达余额** 及缺口。

#### ⑥ 现金流预测
- **折线图**：可切换显示近30/90/180/730天，支持单独开关余额线、收入线、支出线。
- **近两周明细表**：每天的收入、支出、余额，负值行会标红。

#### ⑦ 分类报告（默认折叠）
- 选择一个时间段，勾选要分析的事件说明（如“房租”“工资”），点击 **开始分析**。
- 显示各项合计金额及占比，并以饼图展示。
- 可导出分析结果为 CSV 文件（Excel 可直接打开）。

#### ⑧ 更多工具（默认折叠）
- 可直接设置自定义预警天数，与生存底线中的预警线功能相同。

### 常用操作指南

#### 添加一条每月固定支出（如房租）
1. 展开 **收支计划** → **固定周期计划**。
2. 点击“**+ 添加每月/每周计划**”。
3. 选择周期（如“每月”），输入日期（几号）、金额、类型（必要支出）、说明（如“房租”），可选填起止日期。
4. 点击“保存”，表格和图表立刻更新。

#### 编辑或提前结清一条规则
- 在规则右侧点击 **✏️ 编辑** 按钮，该规则下方会展开编辑区。
- 修改任意字段后点击“保存”，或填写 **提前结清日期** 让该规则在某个日期后不再生效。

#### 查看某一类支出的汇总
1. 展开 **分类报告**，设置分析时间段。
2. 勾选要统计的说明标签（例如只勾选“房租”和“水电”）。
3. 点击 **开始分析**，即可看到合计金额、占比和每笔明细。
4. 点击 **📥 导出报告** 保存为 CSV 文件。

#### 导入之前保存的配置
- 点击顶部 **📂 恢复计划**，选择之前导出的 `.json` 文件，所有设置将自动恢复。

---

### 注意事项
- 所有计算均基于未来 **730 天（约两年）**，超出此范围的数据不会显示。
- 规则中的 **提前结清日期** 优先级高于原始结束日期，用于模拟提前还款、提前退租等场景。
- 如果修改了余额或任何收支，**存钱目标** 和 **风险预警** 会立即自动重算。
